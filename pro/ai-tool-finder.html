How to Use — AI Selection Codex
The AI Selection Codex is a living project memory system for NBD Pro. Every session, decision, directive, and open loop lives here — permanently. Use it to brief a new AI agent, track progress, and never lose context.
1
Synthesis — Your home base
The Synthesis page shows where the project stands right now — current state, execution debt counts, and the locked mission for the next session. Start here every time you open the Codex.
→ Click "Synthesis" in the sidebar
2
Task Registry — Track every open loop
Every task has a priority (Critical / High / Med), an origin session, and an estimated time. Click the checkbox to close a task. Use filters to focus on what's critical. Add new tasks with the "+ Add Task" button.
→ Click "Task Registry" in the sidebar
3
Session Log — Your permanent history
All 11 sessions are logged here. Click any session card to expand it and see the full input vector, output vector, delta log, key decisions, files produced, and open loops. Click "+ Log Session" to add a new one — it auto-increments the session number and version.
→ Click "Session Log" in the sidebar
4
Directives — The laws of this project
Directives are permanent rules established in specific sessions. They cannot be removed — only added to. Use "+ Add Directive" to append a new one tagged to the current session.
→ Click "Directives" in the sidebar
5
Project DNA — The immutable foundation
Core vision, four pillars, all file paths, architecture specs, and non-negotiables. This is what you show a new agent at the start of every session so it understands the project without re-explaining it.
→ Click "Project DNA" in the sidebar
6
Export — Brief your next agent
When you're starting a new AI conversation, export the Codex as a markdown file. Paste it at the beginning of the conversation. The agent will have full context from Session 1 through the current session — no re-explaining required.
→ Click "Export" in the sidebar · Enter next version (e.g. v12.0) · Download
7
Everything saves automatically
All tasks, sessions, and directives you add are saved to your browser's persistent storage. They'll be there the next time you open this page — even after closing the browser.
