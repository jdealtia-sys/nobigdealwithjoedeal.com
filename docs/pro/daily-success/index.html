📱 Add to Your Home Screen for Instant Daily Access
🍎 iPhone / Safari
  1. Tap the Share button ⎙ at the bottom of Safari
  2. Scroll down and tap "Add to Home Screen"
  3. Name it Daily Success → tap Add
  4. Opens full-screen instantly — no browser bar, no login
🤖 Android / Chrome
  1. Tap the ⋮ three-dot menu in the top-right corner
  2. Tap "Add to Home Screen"
  3. Name it Daily Success → tap Add
  4. Tap the new icon on your home screen — launches instantly
💻 Desktop / Chrome
  1. Click the install icon ⊕ in the address bar (right side)
  2. Or click ⋮ menu → More Tools → Create Shortcut
  3. Check "Open as window" → click Create
  4. Bookmark URL: nobigdealwithjoedeal.com/pro/daily-success

Changing Your Theme
1
Tap the 🎨 palette icon in the topbar to open Appearance.
2
Browse by category or search by name (e.g. "batman", "matrix", "lofi").
3
Tap any bubble to apply instantly. Saved automatically across both CRM and Daily Success.
4
All themes and fonts are unlocked — pick any style you like.
Font Pairings
1
Open Appearance → tap Fonts tab for 8 curated pairings.
2
Each pairing sets display, body, and mono fonts across the whole app.
Custom Theme Builder
1
Open Appearance → tap Custom Builder.
2
Pick Background, Surface, Accent, Text, and Muted colors.
3
Apply Preview to see live. Save to keep it (Command plan).
